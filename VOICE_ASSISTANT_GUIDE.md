## 🚀 **How to Use Voice Assistant**

### **For Guests (Index/Landing Page)**

#### **1. Enable Voice**
- Click the **Volume** button (bottom right)
- You'll hear: "Welcome to Royal Springs Resort"
- Voice is now active

#### **2. Start Listening**
- Click the **Microphone** button (green)
- Button shows: "🎤 Listening..." 
- **Say a command** like:
  - "Book a room"
  - "Tell me about dining"
  - "Where are you located?"
  - "Show me food options"
  - "Help"

#### **3. Get Real-Time Response**
- Assistant understands your voice
- Provides natural spoken response
- Shows text summary on screen
- Records command history

#### **4. Record Audio**
- Click the **Record** button (red) to capture audio
- Voice level meter shows audio intensity
- Click again to stop recording
- Perfect for feedback/notes

## 🐛 **Troubleshooting**

### **Voice Not Working?**

**Problem**: Voice commands not recognized
**Solution**: 
1. Check browser supports Web Speech API
2. Ensure microphone permission granted
3. Try refreshing page
4. Clear browser cache

**Problem**: Can't hear responses
**Solution**:
1. Check volume is enabled (Volume button)
2. Check browser volume not muted
3. Check device volume
4. Try different voice voice

**Problem**: Recording won't start
**Solution**:
1. Grant microphone permission
2. Check browser supports MediaRecorder
3. Refresh page
4. Try different browser

### **Images Not Loading?**

**Problem**: Gallery images blurry/pixelated
**Solution**:
1. Check internet connection
2. Clear browser cache
3. Refresh page
4. Check display resolution

**Problem**: Slow image loading
**Solution**:
1. Check internet speed
2. Close other tabs/apps
3. Wait for images to load
4. Try again later

## 📚 **Advanced Features**

### **Command History**

The voice assistant keeps track of:
- Last 10 commands
- Recognition accuracy
- Response time
- User preferences

Access via the voice status card (shows when speaking).

### **Voice Level Meter**

Shows audio input intensity:
- **Low (0-30%)**: Whisper/background noise
- **Medium (30-70%)**: Normal conversation
- **High (70-100%)**: Loud/shouting

Helps optimize microphone positioning.

### **Real-Time Transcription**

Shows:
- **Interim text**: Partial recognition (gray)
- **Final text**: Confirmed command (black)
- **Response**: AI assistant reply (blue)

## 🌟 **Best Practices**

### **For Best Voice Recognition**

1. **Speak clearly** - Enunciate words
2. **Normal pace** - Don't rush
3. **Good lighting** - Helps with visual feedback
4. **Quiet environment** - Reduce background noise
5. **Microphone positioned** - 6-12 inches away
6. **Close doors** - Minimize echo

### **For Best Gallery Experience**

1. **High-quality display** - Appreciate details
2. **Full brightness** - See colors correctly
3. **Desktop/tablet** - Better viewing than mobile
4. **Fast internet** - Images load quicker
5. **Landscape orientation** - More space

# PWA Installation Guide - Royal Springs Resort

## 📱 Install on Mobile Devices

### Android Devices
1. Open the website in Chrome browser
2. Look for the "Install" banner at the top or tap the menu (⋮)
3. Select "Install app" or "Add to Home screen"
4. Confirm installation by tapping "Install"

### iOS Devices (iPhone/iPad)
1. Open the website in Safari browser
2. Tap the Share button (⎋) at the bottom
3. Scroll down and tap "Add to Home Screen"
4. Tap "Add" to confirm installation

### Desktop Devices
1. Open the website in Chrome/Edge browser
2. Look for the install icon (⬇) in the address bar
3. Click the icon and select "Install"
4. Follow the prompts to complete installation

## 📋 Troubleshooting

### If install prompt doesn't appear:
1. Refresh the page and try again
2. Clear browser cache and reload
3. Ensure you're using a supported browser (Chrome, Safari, Edge)
4. Check that you have sufficient storage space

### For iOS users:
- Must use Safari browser (not Chrome/Firefox)
- Ensure "Add to Home Screen" is enabled in Settings
- May need to enable JavaScript in Safari settings

### For Android users:
- Use Chrome browser for best experience
- Ensure Google Play Services are up to date
- Check that website is served over HTTPS
